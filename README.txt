WebChain Order Sync allows WooCommerce store owners to broadcast all completed orders directly to E-Talk's WebChain.

== Description ==
This plugin automatically syncs WooCommerce orders to the WebChain blockchain for fast, secure, and decentralized order tracking.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin via the 'Plugins' menu in WordPress
3. Configure your WebChain API key in WooCommerce > Settings > WebChain
